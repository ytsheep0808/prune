= Prune (プルーン)

* http://wiki.github.com/ytsheep0808/prune

== DESCRIPTION:

Prune(プルーン)はPDF文書をRubyで作成するためのライブラリです。

== FEATURES:

* 日本語をフルサポートしています。
* 文書と表組みの処理に特化しています。

== SYNOPSIS:

  require "prune"
  
  Prune.pdf("test.pdf") do
    preferences do
      description do
        version "1.4"
        title "synopsis"
        author "demo"
      end
    end
  
    page(:A4) do
      
    end
  end

== REQUIREMENTS:

* ruby 1.8

== INSTALL:

* sudo gem install prune
